## 기능
### 계정
회원가입, 로그인
### 팀관리
팀 생성, 팀 참가, 팀 비밀번호 설정
### 품목 관리
품목 등록, 품목 배달시작, 품목 배달완료, 품목 등록시 알림
### 배달 기록
캘린더로 배달 기록 열람
